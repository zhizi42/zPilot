# zPilot
这是一个目前用于给X-Plane11连飞的软件。
目前仅有基础显示飞机的功能。
***
### 优点
* 性能：相比ECHO Pilot：连接时性能提升了114514%，未连接时性能提升了1919810%[^1]
[^1]:均由质子本人测得，不保证准确
### 缺点
* 自研服务端，与其它服务器不兼容。
* 不支持微软模拟飞行2020和Prepar3D。
***
欢迎任何人使用本服务端搭建服务器，也欢迎来官方服务器连飞。

主程序采用Python编写，X-Plane插件采用C艹编写。
